Датчик для избегания препятствий KY-032 (Avoidance Sensor)

Краткое описание

“KY-032” – модуль представляет собой инфракрасный датчик обнаружения препятствий. Устройство имеет передатчик (IR LED), излучающий инфракрасный луч частотой 38 кГц,
который принимается приемником (IR receiver) на плате. При приближении какого-либо предмета к сенсору (близость регулируется потенциометром) на выходе “OUT”
появляется низкий уровень напряжения и включается светодиод на плате модуля.

Код программы, которая работает с этим модулем


digitalWrite( enablePin, HIGH);     //Включить внутренний сигнал 38 кГц
microDelay( 210);                   //Задержка 210µs(8 пульсов по 38kHz)
	if( digitalRead( outputPin))        //Если Output детектора равен HIGH
	{
		objectDetect = false;           //Препятсвия не было найдено
	}
	else                                //Но если Output равен LOW,
	{
		microDelay( 395);               //Опять ждать 15 пульсов
		if( digitalRead( outputPin))    //Если Output теперь равен HIGH
		{
			objectDetect = false;       //Препятсвия не было найдено
		}
		else                            //Если Output до сих пор LOW
		{
			objectDetect = true;        //Препятсвие било обнаружено
		}
	}
digitalWrite( enablePin, LOW);      //Выключить внутренний сигнал 38kHz

Лазерный излучатель Laser Emiter (KY-008 Red Laser Head Transmitter Module)

Краткое описание

“KY-008” – лазерный модуль имеет длину волны 650 нм и может использоваться с Arduino.
Модуль является одним из самых часто встречаемых модулей в “Arduino”. Объеденённый с другими компонентами,
он может использоваться для создания лазерной растяжки или оптического эффекта. “KY-008” особенно удобен в использовании,
так как может напрямую питаться от выходного контакта “Arduino”, поскольку он потребляет всего лишь 30мА от процессора, выход которого, в свою очередь, расчитан на 40мА.

Код программы, которая работает с этим модулем


int laserPin = 10;
void setup ()
{
pinMode (laserPin, OUTPUT); // define the digital output interface 13 feet
}
void loop () {
laserEmiter(laserPin,15,5);
}

digitalWrite (laserPin, HIGH); // Turn Laser On
delay (timeToBurn);// delay while it burns
digitalWrite (laserPin, LOW); // Turn Laser Off
delay (timeToRest);//delay till new cicle
}

KY-018 Фоторезистор (KY-018 Photoresistor Module)

Краткое описание

“KY-018″ — модуль содержит фоторезистор (датчик освещенности), который может применяться для измерения интенсивности света или определения его наличия/отсутствия.
При отсутствии света сопротивление фоторезистора большое и доходит до 1 мОм, а при его освещении падает до нескольких Ом.
Фоторезисторы – это полупроводниковые фоточувствительные устройства (сенсоры), которые имеют высокую чувствительность,
быстрый отклик, устойчивые спектральные характеристики и широко применяются в видеокамерах,
в садовых светильниках на солнечных батареях, датчиках освещенности, ночниках и других устройствах контроля освещения.

Код программы, которая работает с этим модулем

            
int sensorPin = 2; //define analog pin 2
int value = 0;
void setup() {
	Serial.begin(9600);
}
void loop() {
	value = analogRead(sensorPin);
	Serial.println(value, DEC); // light intensity
	// high values for bright environment
	// low values for dark environment
	delay(100);
}
